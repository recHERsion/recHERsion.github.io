About recHERsion

RecHERsion is the annual flagship event organised by ACM-W NITK. This year it is scheduled to be held on 16th of December. It is a coding contest exclusively for undergraduate girls that lasts for around 2.5 - 3 hours. The contest contains 4-5 CP/DSA coding questions of varying difficulties. Questions and sample test cases will be given, and participants are expected to come up with a proper working code which passes all the test cases to solve these questions.

Frequently Asked Questions

Here are some of the most asked questions.

  • Who can participate in this event?

    All girls who are currently doing their Undergraduate studies from all over the world are eligible to participate.

  • What are the prerequisites required for the contest?

    Basic coding skills in any language of your preference is sufficient. Since it's a CP contest, having an idea about different data structures and algorithms will be helpful. However, don't worry about the prerequisites as the event is all about learning and loving to code more, so it's totally fine if you're new!

  • What are the cash prizes that are going to be awarded?

    As stated above

  • What are the type of questions which will be asked?

    4-5 CP/DSA questions will be asked.

  • What mode will the event be conducted?

    The event will be conducted online, on the HackerRank platform.
